# Day 2 – Salesforce Platform Basics

## 1. What is Salesforce Platform?

The Salesforce Platform is a cloud-based platform used to build and manage business applications. It allows developers and admins to create apps, automate processes, and manage data efficiently.

---

## 2. Core Concepts

### App
An App is a collection of objects and features used for a business process.

Example: Sales App (Leads, Accounts, Opportunities)

---

### Object
An Object is like a database table that stores data.

Types:
- Standard Objects (Account, Contact)
- Custom Objects (User-created)

Example: Student Object, Product Object

---

### Tab
A Tab is a UI element used to access objects or apps.

Example: Accounts Tab, Contacts Tab

---

## 3. Configuration vs Coding

| Configuration (No-Code) | Coding (Pro-Code) |
|-------------------------|------------------|
| Click-based setup       | Programming (Apex, LWC) |
| Fast development        | Advanced logic |
| Easy for admins         | Used by developers |

---

## 4. System Design Example

### College Admission System

**App:** Admission Management App

**Objects:**
- Student (Name, Marks, Email)
- Application (Course, Status)
- Course (Fee, Duration)

**Flow:**
Student → Application → Review → Approval → Notification

---

## Conclusion

Salesforce Platform supports both:
- No-code/low-code development
- Pro-code development using Apex, LWC, APIs
